Mulai dalam 4 Langkah
Dari nol sampai bisa dipakai tim dalam kurang dari 30 menit
1
Jalankan setup.sql di Supabase
Buka Supabase → SQL Editor → paste isi file setup.sql → klik Run. Tabel, RLS, dan trigger otomatis terbuat.
2
Aktifkan Realtime di Supabase
Table Editor → tabel konsumen → ikon ⚡ Realtime → Enable.
3
Isi konfigurasi di index.html
Cari 2 baris ini dan ganti dengan URL & Key dari Supabase → Project Settings → API:
const SUPABASE_URL = 'https://xxxx.supabase.co';
const SUPABASE_ANON_KEY = 'eyJ...';
4
Upload ke Vercel atau Netlify
Drag & drop folder ke vercel.com atau netlify.com → dapat URL untuk dibagikan ke tim. Gratis.
