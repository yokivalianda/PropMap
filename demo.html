Semua yang Dibutuhkan Tim Marketing
Satu aplikasi untuk seluruh proses — dari pertama kali konsumen datang hingga akad selesai
⚡
Real-time Sync
Data tersinkronisasi ke semua HP tim secara instan via WebSocket. Satu konsumen diupdate, semua marketing langsung lihat.
👥
Multi-user & Role
Login per marketing. Admin lihat seluruh data tim, Marketing hanya lihat data konsumen miliknya sendiri.
📲
Install di HP (PWA)
Progressive Web App — install di Android & iPhone tanpa App Store. Tampil seperti aplikasi native.
📁
Checklist Berkas Dinamis Baru
Item checklist bisa ditambah, diedit, dan dihapus sesuai kebutuhan. Upload foto dokumen langsung ke cloud storage.
📅
Kalender Follow-up Baru
Jadwalkan follow-up konsumen di kalender. Pengingat otomatis H-1 dan hari-H saat waktu follow-up tiba.
📊
Grafik & Laporan Baru
Chart distribusi pipeline, tren penjualan 6 bulan, dan sumber leads. Filter per periode. Export ke PDF & CSV.
📥
Import Excel / CSV Baru
Upload data konsumen dari file Excel atau CSV sekaligus. Sistem otomatis mendeteksi nama kolom yang berbeda-beda.
🔔
Pengingat Otomatis
Alert follow-up, berkas kurang, dan DP belum selesai. Notifikasi muncul langsung di header aplikasi.
🏆
Ranking Tim
Laporan performa dan ranking seluruh anggota tim marketing untuk Admin. Lihat siapa yang paling produktif.
🌙
Tema Gelap & Terang
Toggle tema kapan saja dari header atau pengaturan. Tersimpan otomatis, mengikuti preferensi sistem HP.
🔐
Keamanan Row-Level
Row Level Security di PostgreSQL memastikan data marketing tidak bisa diakses marketing lain — di level database.
💻
Tampilan Desktop Baru
Sidebar navigasi di desktop dengan layout 2 panel. Konten terpusat maksimal 800px, tidak melebar di layar lebar.
